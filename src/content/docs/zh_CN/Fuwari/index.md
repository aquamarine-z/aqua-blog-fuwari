欢迎来到 **Fuwari 教学项目**！

这套独家教程汇集了对本框架进行底层魔改与结构升级后的所有核心功能解说。你可以通过左侧的展开菜单，逐步深入了解这套强大的知识库与博客系统的高级运作原理。

**本指南的内容涵盖：**

- 🛠️ **站长配置指南 (Site Config)**：全方位解析 `config.ts` 中的各类设置，带你掌控站点的基础参数。
- 📖 **优雅发布指南**：详述前置元数据（Frontmatter）的进阶用法，让你优雅地发布各类文章。
- 🌳 **Docs 魔法目录树**：介绍文档系统下无限嵌套物理文件夹映射与 `index.md` 的高级控制权。
- 📝 **Blog 与 Spec 的奥秘**：解读扁平化博客流与孤立静态单页（Spec）的不同战略定位。
- 🌐 **全方位多语言 (i18n)**：深入了解防串台隐身机制与平滑的区域语言降级回退策略。
- 💬 **Giscus 评论区指南**：如何优雅地接入 Giscus 评论系统并进行多语言支持。
- 🤝 **友链配置基础**：基础的朋友链接数据格式及头像配置指南。
- 🚀 **深度定制区**：探索 MDX 组件库、响应式断点、动态主题色等核心源码级别的进阶魔改技巧。

希望这份详尽的教程能帮助你将个人站点打磨至极致！
