# 📚 第五部分：成本管理与控制 (Cost Management)

### 1. 核心成本术语定义

* **Direct Costs (直接成本)**：可完全归属于单一项目工作的成本（如：项目组专属软件开发人员的工资、采购的服务器等）。
* **Indirect Costs (间接成本)**：由多个项目或整个公司共同分摊、难以直接归属于某个具体项目的成本（如：公司办公室租金、前台行政的工资、技术支持部门的折旧等）。
* **Sunk Cost (沉没成本)**：Money that has been spent in the past. When deciding whether to continue a project, sunk costs **should be forgotten**.（在进行项目投资或止损决策时，必须排除历史已支出成本的干扰，只看未来的投入产出比。不能因为“已经花了100万”而盲目追加投资一个注定失败的项目。）
* **Contingency Reserves (应急储备)**：Budget within the cost baseline that is allocated for identified risks which are accepted and for which contingent or mitigation responses are developed ("known-unknowns").（针对已识别、已分析风险的专项备用金（“已知-未知”），包含在项目的**成本基准（Cost Baseline）**中，项目经理有权直接动用。）
* **Management Reserves (管理储备)**：An amount of the project budget withheld for management control purposes and which are released for unforeseen work that is within the scope of the project ("unknown-unknowns").（针对未预料到但属于范围内的风险备用金（“未知-未知”），**不包含在成本基准内**，动用时必须经过高级管理层的正式变更审批。）

---

### 2. 估算类型与精度 (Estimating Types)

* **Rough Order Magnitude (ROM, 粗略估算)**：Ballpark estimate, 精度通常在 **-50% 到 +100%**。在项目早期、需求极不明确时使用，作为快速决策的依据。
* **Budgetary Estimate (预算估算)**：精度在 **-10% 到 +25%**。主要用于公司年度资金计划分配或向高层申请项目的预算。
* **Definitive Estimate (确定性估算)**：精度在 **-5% 到 +10%**（或 -10% 到 +15%）。在项目计划完善、工作包细化后进行，用于采购及制定成本绩效基准。
