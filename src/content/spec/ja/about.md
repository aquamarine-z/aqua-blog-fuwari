# 当サイトについて

Aquamarineの個人ブログへようこそ！このサイトは、強力な[Astro Fuwari](https://github.com/saicaca/fuwari)フレームワークに基づいて構築されており、その優れたオリジナルデザインを基に、より包括的で個別のニーズに合わせた個人ナレッジベースおよびショーケースプラットフォームを作成するために、一連の深いカスタマイズと機能強化を行っています。

## ✨ コア強化機能

### 1. 🌐 強力な国際化 (I18n) サポート
グローバル化が進む今日、言語の壁を越えたコミュニケーションと技術共有は特に重要です。オリジナルバージョンを基に、国際化ルーティングとレンダリングロジックを完全にリファクタリングしました：
- **シームレスな多言語切り替え**：任意の言語パスの記事とドキュメントをサポートします。異なる言語バージョンを切り替える際、現在の記事に対応するターゲット翻訳バージョンに正確に移動します。
- **スマートなフォールバックメカニズム**：ターゲット言語の翻訳がまだ提供されていない場合、システムは自動的にメイン言語のコンテンツを表示し、親切なトップヒントを提供します。これにより、読者が404ページに遭遇するのを防ぎます。
- **柔軟なディレクトリ構造**：記事を特定の階層に保存しなければならないという制限を解除しました。ルートディレクトリや各言語専用フォルダでMarkdownコンテンツを自由に管理できます。

### 2. 📚 新しいドキュメント (Docs) システム
タイムライン駆動の通常のブログ記事に加えて、このサイトには独立した新しい**Docsシステム**が導入されています：
- **ツリー型マルチレベルナビゲーション**：フォルダ構造に基づくツリーディレクトリをサイドバーに自動生成します。チュートリアルシリーズ、体系的な技術ノート、連載コンテンツの執筆に最適です。
- **知識体系の蓄積**：断片化された日常のブログと体系的なハードコアな知識を完全に分離し、長文の技術ドキュメントにより集中して没入できる読書体験を提供します。
- **包括的な多言語対応**：DocsシステムもグローバルなI18nルーティングとフォールバックメカニズムを完全に継承しています。

### 3. 🎵 グローバルBGMプレイヤー
雰囲気のある没入型の読書体験を提供するために、ウェブサイトにグローバルなBGM再生機能を追加しました：
- **シームレスな再生**：SwupベースのSPAスムーズルーティングメカニズムにより、記事、カテゴリ、タグページ間をどのように移動しても、BGMの再生が途切れることはありません。
- **パーソナライズされたカスタマイズ**：将来的にはカスタムプレイリストをサポートし、すべての訪問者の読書時間に音楽を添える予定です。

### 4. 📱 モバイルTOCの最適化
長文記事やドキュメントにおいて、サイドバーの目次（TOC）は読書体験を向上させる鍵です。モバイルデバイスにおけるオリジナル版の不便さを解消するため、専用のレスポンシブ最適化を行いました：
- **モバイルフローティング目次**：スマートフォンやタブレットなどの小さな画面では、記事のアウトラインが自動的に折りたたまれ、エレガントなフローティングボタンになります。
- **ワンタップアクセス**：長文を読んでいる最中にいつでも目次を呼び出し、関心のあるセクションにすばやくジャンプできるため、長いページを何度もスクロールする手間が省け、モバイルユーザーの検索効率が大幅に向上します。

### 5. 🗂️ BlogとDocsの独立したアーカイブシステム
コンテンツの検索をより直感的にするため、アーカイブ（Archive）ページを再構築・拡張しました：
- **独立したタイムライン**：Blog記事とDocsドキュメントそれぞれに独立したアーカイブページを設け、互いに干渉しないようにしました。
- **明確なコンテンツ整理**：過去のエッセイを振り返る場合でも、特定のチュートリアルを探す場合でも、それぞれの専用アーカイブページで一目で確認できます。

### 6. 🤝 リンク（Friends）システム
ネット上での出会いも何かの縁です。新しく洗練されたリンク（Friends）ページを追加しました：
- **専用のフレンドカード**：統一されたカスタマイズUIカードを使用して、ブロガーの友人たちのアバター、ニックネーム、紹介文、およびURLを美しく表示します。
- **簡単な拡張と交流**：リンクのカテゴリ分けをサポートし、サイトをより人情味のあるものにするだけでなく、お互いのサイトを訪問し合うのにも便利です。
