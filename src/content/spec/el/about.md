# Σχετικά με αυτόν τον ιστότοπο

Καλώς ήρθατε στο προσωπικό blog του Aquamarine! Αυτός ο ιστότοπος βασίζεται στο ισχυρό πλαίσιο [Astro Fuwari](https://github.com/saicaca/fuwari). Με βάση τον εξαιρετικό αρχικό σχεδιασμό του, πραγματοποιήθηκε μια σειρά λεπτομερών προσαρμογών και βελτιώσεων λειτουργιών, με στόχο τη δημιουργία μιας πιο ολοκληρωμένης και εξατομικευμένης βάσης γνώσεων και πλατφόρμας παρουσίασης.

## ✨ Βασικά χαρακτηριστικά βελτίωσης

### 1. 🌐 Πιο ισχυρή υποστήριξη Διεθνοποίησης (I18n)
Στη σημερινή παγκοσμιοποιημένη εποχή, η διαγλωσσική επικοινωνία και η ανταλλαγή τεχνολογίας έχουν γίνει ιδιαίτερα σημαντικές. Αναδιαμόρφωσα πλήρως τη λογική δρομολόγησης και απόδοσης της διεθνοποίησης:
- **Απρόσκοπτη εναλλαγή πολλαπλών γλωσσών**: Υποστηρίζει άρθρα και έγγραφα σε οποιεσδήποτε διαδρομές γλωσσών. Κατά την εναλλαγή μεταξύ διαφορετικών γλωσσικών εκδόσεων, θα εντοπίζεται αυτόματα η σωστή μετάφραση-στόχος του τρέχοντος άρθρου.
- **Έξυπνος μηχανισμός υποβάθμισης (Fallback)**: Όταν η γλώσσα-στόχος δεν έχει ακόμη μετάφραση, το σύστημα υποβαθμίζεται ομαλά, εμφανίζοντας το περιεχόμενο της κύριας γλώσσας με μια φιλική ειδοποίηση στην κορυφή, διασφαλίζοντας ότι οι αναγνώστες δεν θα αντιμετωπίσουν ποτέ το πρόβλημα του να μη βρίσκουν μια σελίδα.
- **Ευέλικτη δομή καταλόγου**: Καταργήθηκε ο περιορισμός που υποχρέωνε τα άρθρα να αποθηκεύονται σε συγκεκριμένα επίπεδα. Τώρα μπορείτε ελεύθερα να διαχειριστείτε το περιεχόμενο Markdown στον ριζικό κατάλογο ή σε οποιονδήποτε φάκελο ειδικό για κάποια γλώσσα.

### 2. 📚 Νέο σύστημα εγγράφων (Docs)
Εκτός από τα συνηθισμένα άρθρα ιστολογίου (Blog), σε αυτόν τον ιστότοπο εισήχθη επίσης ένα εντελώς νέο **σύστημα εγγράφων Docs**:
- **Δενδρική πλοήγηση πολλών επιπέδων**: Δημιουργεί αυτόματα έναν κατάλογο δενδρικής μορφής στην πλευρική γραμμή με βάση τη δομή των φακέλων, ιδανικό για τη συγγραφή σειρών μαθημάτων, συστημικών τεχνικών σημειώσεων ή σειριακού περιεχομένου.
- **Συγκέντρωση γνώσεων**: Διαχωρίζει τέλεια τα καθημερινά άρθρα του ιστολογίου από τις συστηματικές και εις βάθος γνώσεις, προσφέροντας μια πιο συγκεντρωμένη και καθηλωτική εμπειρία ανάγνωσης για μακροσκελή τεχνικά έγγραφα.
- **Ολοκληρωμένη προσαρμογή πολλαπλών γλωσσών**: Το σύστημα Docs κληρονομεί επίσης την καθολική δρομολόγηση I18n και τον μηχανισμό Fallback.

### 3. 🎵 Παγκόσμια συσκευή αναπαραγωγής μουσικής (BGM)
Για να προσφέρω μια πιο ατμοσφαιρική και καθηλωτική εμπειρία ανάγνωσης, πρόσθεσα μια λειτουργία αναπαραγωγής μουσικής φόντου σε όλο τον ιστότοπο:
- **Απρόσκοπτη αναπαραγωγή**: Χάρη στον ομαλό μηχανισμό δρομολόγησης της εφαρμογής (SPA) που βασίζεται στο Swup, η μουσική υποβάθρου δεν θα διακοπεί ποτέ απροσδόκητα, ανεξάρτητα από το πώς πλοηγείστε σε άρθρα, κατηγορίες ή σελίδες ετικετών.
- **Εξατομικευμένη προσαρμογή**: Στο μέλλον, θα υποστηρίζονται περισσότερες προσαρμοσμένες λίστες αναπαραγωγής, ώστε η μουσική να συνοδεύει τον χρόνο ανάγνωσης κάθε επισκέπτη.

### 4. 📱 Βελτιστοποιημένο TOC για κινητά
Για μακροσκελή άρθρα και έγγραφα, ο πλευρικός κατάλογος (TOC) είναι το κλειδί για τη βελτίωση της εμπειρίας ανάγνωσης. Ως απάντηση σε πιθανές δυσκολίες της αρχικής έκδοσης σε κινητές συσκευές, πραγματοποιήσαμε ειδικές βελτιστοποιήσεις:
- **Αιωρούμενος κατάλογος σε κινητά**: Σε μικρότερες οθόνες όπως τηλέφωνα ή tablet, ο κατάλογος άρθρων συμπτύσσεται αυτόματα και μετατρέπεται σε ένα κομψό αιωρούμενο κουμπί.
- **Πρόσβαση με ένα κλικ**: Όταν διαβάζουν μεγάλα κείμενα, οι επισκέπτες μπορούν να εμφανίσουν τον κατάλογο ανά πάσα στιγμή και να μεταβούν γρήγορα στο κεφάλαιο που τους ενδιαφέρει, μειώνοντας την περιττή κύλιση και αυξάνοντας την απόδοση ανάγνωσης.

### 5. 🗂️ Σύστημα αρχειοθέτησης με διαχωρισμό Blog και Docs
Για να κάνουμε την εύρεση περιεχομένου πιο καθαρή και διαισθητική, ανακατασκευάσαμε και επεκτείναμε τη σελίδα Αρχείου (Archive):
- **Ανεξάρτητα χρονοδιαγράμματα αρχειοθέτησης**: Δημιουργήθηκαν χωριστές σελίδες αρχειοθέτησης για τα άρθρα του Blog και τα έγγραφα Docs, ώστε να μην παρεμβαίνει το ένα στο άλλο.
- **Καθαρή οργάνωση περιεχομένου**: Είτε ανασκοπείτε προηγούμενα δοκίμια είτε ψάχνετε για συγκεκριμένα εκπαιδευτικά προγράμματα, όλα παρουσιάζονται ξεκάθαρα στις δικές τους σελίδες αρχειοθέτησης.

### 6. 🤝 Σύστημα συνδέσμων φίλων
Οι διαδικτυακές συναντήσεις είναι μοίρα. Πρόσθεσα μια όμορφη σελίδα συνδέσμων με φίλους (Friends):
- **Αποκλειστικές κάρτες φίλων**: Χρησιμοποιεί ομοιόμορφες κάρτες UI για να προβάλλει τα άβαταρ, τα ψευδώνυμα, τις υπογραφές και τις διευθύνσεις των φίλων bloggers, με έναν όμορφο και κομψό σχεδιασμό.
- **Εύκολη επέκταση και αλληλεπίδραση**: Υποστηρίζει κατηγοριοποίηση συνδέσμων φίλων, κάτι που όχι μόνο προσθέτει χαρακτήρα στον ιστότοπο, αλλά διευκολύνει επίσης την αμοιβαία αλληλεπίδραση.
